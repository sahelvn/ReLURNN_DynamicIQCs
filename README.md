# Dynamic IQCs for Repeated ReLU

This repository contains MATLAB/CVX implementations accompanying the paper:

> [**Dynamic Integral Quadratic Constraints for Repeated ReLU**](https://arxiv.org/abs/2603.17137)  
> Sahel Vahedi Noori, Bin Hu, Geir Dullerud, and Peter Seiler

The codes implement the dynamic IQC conditions proposed in the paper for stability and induced-l2 gain analysis of discrete-time feedback interconnections with repeated ReLU nonlinearities.


# Requirements

The following software packages are required:

- [MATLAB](https://www.mathworks.com/)
- [CVX](https://cvxr.com/)
- [MOSEK](https://www.mosek.com/)
